# オーディオファイル配置ガイド

このディレクトリには、ゲームで使用するBGMと効果音を配置します。

## ディレクトリ構造

```
audio/
└── bgm/          # BGM（背景音楽）
    ├── title.mp3              # タイトル/マップ画面
    ├── roulette.mp3           # 目的地ルーレット
    ├── dice_wait.mp3          # サイコロ待機
    ├── flying.mp3             # 飛行中（エンジン音）
    ├── attraction.mp3         # 名所/星マス（明るく軽快なピアノ）
    ├── calm.mp3               # アート/グルメ/人マス（癒しのピアノ）
    ├── trouble.mp3            # トラブルマス（暗く重いピアノ）
    ├── arrival.mp3            # 目的地到着
    └── ending.mp3             # エンディング
```

**重要**: ファイルが存在しない場合、Web Audio APIによる自動生成音が使用されます。
- 飛行機のエンジン音: リアルなブラウンノイズ + ドップラー効果
- その他のBGM: 音楽理論に基づいたメロディ生成

## BGMの使用シーン

### title.mp3
- **使用場面**: タイトル画面、マップ画面
- **推奨**: 明るく軽快なメロディ、冒険心をくすぐる曲調
- **長さ**: ループ再生可能な1〜2分程度
- **フォーマット**: MP3, 128kbps以上

### roulette.mp3
- **使用場面**: 目的地ルーレット
- **推奨**: ワクワク感のあるメロディ、期待感を高める曲調
- **長さ**: ループ再生可能な30秒〜1分程度
- **フォーマット**: MP3, 128kbps以上

### dice_wait.mp3
- **使用場面**: サイコロ待機画面
- **推奨**: 期待感のあるメロディ、緊張感のある曲調
- **長さ**: ループ再生可能な30秒〜1分程度
- **フォーマット**: MP3, 128kbps以上

### flying.mp3
- **使用場面**: 飛行機での移動中
- **推奨**: リアルな飛行機のエンジン音、ジェットエンジンの通過音
- **長さ**: ループ再生可能な30秒〜1分程度
- **フォーマット**: MP3, 128kbps以上
- **自動生成**: ブラウンノイズ + ドップラー効果で8秒周期のエンジン通過音を生成

### attraction.mp3
- **使用場面**: 名所マス、星マス
- **推奨**: 明るく軽快なピアノ曲、テンポ100-120BPM
- **長さ**: ループ再生可能な1〜2分程度
- **フォーマット**: MP3, 128kbps以上
- **自動生成**: 明るい三和音メロディ（C-E-G-C5-E5系）

### calm.mp3
- **使用場面**: アートマス、グルメマス、人マス
- **推奨**: 癒される優しいピアノ曲、テンポ60-90BPM
- **長さ**: ループ再生可能な1〜2分程度
- **フォーマット**: MP3, 128kbps以上
- **自動生成**: 穏やかなメロディ（C-E-F-G-A-G-F-E-D-C系）

### trouble.mp3
- **使用場面**: トラブルマス
- **推奨**: 重低音の暗く物悲しいピアノ曲、短調（C minor等）
- **長さ**: ループ再生可能な1〜2分程度
- **フォーマット**: MP3, 128kbps以上
- **自動生成**: 暗いマイナーコード（C-Eb-F-G-Ab系）、サウトゥース波形

### arrival.mp3
- **使用場面**: 目的地到着時
- **推奨**: 明るいファンファーレ、達成感のある曲調
- **長さ**: 5〜10秒程度（ループなし）
- **フォーマット**: MP3, 128kbps以上
- **自動生成**: 上昇するメロディ（C-E-G-C5-E5-G5）

### ending.mp3
- **使用場面**: ゲーム終了、結果発表
- **推奨**: 華やかなエンディング曲、祝福感のある曲調
- **長さ**: 10〜20秒程度（ループなし）
- **フォーマット**: MP3, 128kbps以上
- **自動生成**: 明るい上昇メロディ（全音階）

## フリー音源サイトの推奨

- **Freesound**: https://freesound.org/
- **DOVA-SYNDROME**: https://dova-s.jp/
- **魔王魂**: https://maou.audio/
- **効果音ラボ**: https://soundeffect-lab.info/

## 注意事項

1. 使用する音源のライセンスを必ず確認してください
2. 商用利用が可能な音源を選んでください
3. クレジット表記が必要な場合は、適切に表示してください
4. ファイルサイズは各ファイル1MB以下を推奨します
5. ブラウザの自動再生ポリシーに注意してください（ユーザーインタラクション後に再生）
